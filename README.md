\# Unreal Combat AI Demo



A focused Unreal Engine C++ gameplay systems demo built to showcase combat AI, health/damage architecture, enemy states, and gameplay debugging tools.



This project is being developed as a personal portfolio piece for gameplay programming and technical game development roles.



\## Project Goals



The goal is to create a small but polished gameplay systems demo rather than a full game. The demo focuses on clear, readable systems implemented mainly in C++, with Blueprint used for presentation and asset setup.



\## Planned Features



\- Modular health and damage component

\- Player melee attack system

\- Enemy detection and aggro behavior

\- Enemy chase and attack states

\- Death state and combat feedback

\- Health UI

\- Debug visualization for testing AI behavior



\## Tech Stack



\- Unreal Engine 5.6

\- C++

\- Blueprint

\- Visual Studio 2022

\- Git / GitHub



\## Current Status



Project setup is complete. The first gameplay system, a reusable Health Component, is currently being implemented.



\## Portfolio Focus



This project is designed to demonstrate:



\- Unreal Engine C++ gameplay programming

\- Component-based gameplay architecture

\- AI state logic

\- Combat readability and feedback

\- Clean project organization

\- Iterative development with version control
